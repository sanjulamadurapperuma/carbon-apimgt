AWS Lambda (Individual)

Retrieve the ARNs of AWS Lambda functions

GET https://apis.wso2.com/api/am/publisher/v1.0/apis/{apiId}/amznResourceNames

This operation can be use to retrieve ARNs of AWS Lambda function for a given AWS credentials.

OAuth 2.0 Scope

apim:api_view

Examples

Request
GET https://localhost:9443/api/am/publisher/v1.0/apis/a8f0fb89-edf9-484c-b937-036b088f287d/amznResourceNames HTTP 1/1 Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8
Response
HTTP 1/1 200 OK Content-Type: application/json { "count" : 2, "list" : [ "arn:aws:lambda:us-west-2:123456789012:function:my-function1", "arn:aws:lambda:us-west-2:123456789012:function:my-function2" ] }
CURL
curl -k -v -X GET 'https://localhost:9443/api/am/publisher/v1.0/apis/a8f0fb89-edf9-484c-b937-036b088f287d/amznResourceNames' -H 'Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8'
